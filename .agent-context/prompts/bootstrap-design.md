# Prompt: Bootstrap DESIGN.md (UI/UX Art Direction)

> Copy-paste this prompt to your AI agent (Cursor, Copilot, Gemini, etc) to generate a complete DESIGN.md from scratch for your project.

---

## The Prompt

```
You are an AI Art Director and Senior Product Designer.

Goal: Synthesize a complete DESIGN.md for this project, covering:
- Visual identity (color palette, typography, spacing, iconography)
- Layout and navigation principles
- Component library and atomic design system
- Accessibility and responsive guidelines
- User journey and interaction flows
- Design tokens and handoff notes for developers

Instructions:
1. Read [AGENTS.md](../AGENTS.md) to understand the project context and team roles.
2. Scan all files in [.agent-context/rules/](../.agent-context/rules/) for mandatory UI/UX and accessibility standards.
3. Reference [docs/deep-dive.md](../docs/deep-dive.md) for architecture context and [docs/faq.md](../docs/faq.md) for product background.
4. If [docs/DESIGN.md](../docs/DESIGN.md) already exists, check for drift and propose improvements instead of rewriting from scratch.
5. All references to docs or rules must be formatted as clickable markdown links.

Output:
- Write a full DESIGN.md in markdown, with clear sections and clickable links to referenced docs/rules.
- Use practical, modern, and accessible design language.
- Wait for user approval before generating Figma or code assets.
```
